# Fitnesscomapinion

## Funcionalidades
- Uso de ViewModel
- Navegação com NavController e NavHost
- Lista com LazyColumn
- Suporte para tema dark e light
- Botões e imagens
- Fonte personalizada
- Top Bar
- Animação simples

## Como Executar
Clone o repositório e abra no Android Studio. Compile e execute o projeto no emulador ou dispositivo.
